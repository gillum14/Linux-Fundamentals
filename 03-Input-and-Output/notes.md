## Input and Output

### echo

The `echo` command prints text to the terminal.

```bash
echo "Hello World"
```

Output:

```
Hello World
```

Common uses:

- Verify variables
- Print messages
- Test scripts
- Output text

---

### expr

The `expr` command evaluates arithmetic expressions.

```bash
expr 32 - 8
```

Output:

```
24
```

Example:

```bash
expr 3500 \* 12
```

Output:

```
42000
```

Common uses:

- Basic shell calculations
- Integer arithmetic
- Script automation

---

### clear

Clears the terminal window.

```bash
clear
```
