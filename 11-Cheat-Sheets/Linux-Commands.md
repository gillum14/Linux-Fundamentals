# Linux Commands Cheat Sheet

## Navigation

| Command | Description |
|----------|-------------|
| `pwd` | Display current working directory |
| `ls` | List directory contents |
| `ls -l` | Long listing format |
| `ls -la` | Show hidden files |
| `cd directory` | Change directory |
| `cd ..` | Move up one directory |
| `cd ~` | Go to home directory |
| `cd /` | Go to root directory |

---

## File & Directory Management

| Command | Description |
|----------|-------------|
| `mkdir directory` | Create directory |
| `rmdir directory` | Remove empty directory |
| `touch file.txt` | Create empty file |
| `cp source destination` | Copy files |
| `mv source destination` | Move or rename files |
| `rm file.txt` | Delete file |

---

## Viewing Files

| Command | Description |
|----------|-------------|
| `cat file.txt` | Display entire file |
| `head file.txt` | Display first 10 lines |
| `head -n 5 file.txt` | Display first 5 lines |
| `tail file.txt` | Display last 10 lines |
| `less file.txt` | View file one page at a time |

---

## Searching & Filtering

| Command | Description |
|----------|-------------|
| `grep "text" file.txt` | Search file for text |
| `find /path -name filename` | Locate files |
| `locate filename` | Search indexed files |
| `which command` | Show command location |
| `whereis command` | Locate binary, source, and man page |

---

## User & Permission Management

| Command | Description |
|----------|-------------|
| `chmod` | Change permissions |
| `chown` | Change ownership |
| `chgrp` | Change group ownership |
| `useradd` | Create user |
| `usermod` | Modify user |
| `userdel` | Delete user |
| `groupadd` | Create group |
| `groupdel` | Delete group |
| `sudo` | Execute as superuser |

---

## Useful Terminal Commands

| Command | Description |
|----------|-------------|
| `clear` | Clear terminal |
| `history` | View command history |
| `echo` | Print text |
| `expr` | Integer calculations |
| `nano file.txt` | Edit file |

---

## Command Chaining

| Operator | Purpose |
|----------|----------|
| `|` | Pipe output into another command |
| `>` | Redirect output to file |
| `>>` | Append output to file |
| `<` | Read input from file |

---

## Helpful Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + C` | Stop current process |
| `Ctrl + L` | Clear terminal |
| `Tab` | Auto-complete |
| `↑` | Previous command |
| `Ctrl + A` | Beginning of line |
| `Ctrl + E` | End of line |
